StackOBot Level

Idle to walk to run animation. Faint dust clouds left behind as player runs.
Player drops orbs which NPC travels to and picks up.
Pressure plate opens large door. Opening/closing door causes the world to shake.
Clean blueprint graphs.
❮ ❯

Project Status: Complete
Project Duration: 1 week
Programs used: Unreal Engine 5.1
Logic: Blueprinting

I created a simple level in Unreal Engine 5.1 following a course from the Epic Developer Community: "Your First Game in Unreal Engine 5." The player is trapped in a valley, along with a fellow StackOBot, with the only exit being behind a giant door.

I enjoyed creating my own landscape and adjusting the provided materials to make the level my own. This project gave me a great grasp on blueprinting logic in UE5, and it was also a good introduction to animating characters.
