Pong

Pong gameplay. AI paddle is on the right.
Score is updated when a goal is made.
❮ ❯

Project Status: Complete
Project Duration: 4 days
Software used: Unreal Engine 4.27
Logic: C++ and blueprinting

This Pong game was created while following a course from Pluralsight. It was a great introduction to implementing game logic using C++ and blueprinting together. The project files included a simple Pong game skeleton, and I finished the project by implementing the logic for the ball, boundaries, and paddles. I also added a simple text UI element just outside the board that keeps the player updated on the score.
